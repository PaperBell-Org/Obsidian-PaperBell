# PaperBell Note Conventions

## Literature Notes
- Must include: title, authors, year, doi, tags
- Body structure: Summary, Key Findings, Methodology, Notes

## Scholar Notes
- Must include: name, institution, research_areas
- Auto-generated from file creation in Persons/Scholars

## Institution Notes
- Must include: name, country, type
- Can be auto-generated via PaperBell settings
