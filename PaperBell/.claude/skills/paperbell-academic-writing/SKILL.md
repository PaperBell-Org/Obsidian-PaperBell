# PaperBell Academic Writing

When working in this PaperBell vault, follow these conventions:

## Vault Structure
- `Inputs/Zotero` — Literature notes imported from Zotero
- `Persons/Scholars` — Scholar profiles
- `Locations/Institutes` — Institution notes
- `Projects/` — Research project folders

## Writing Guidelines
- Use academic tone and precise terminology
- Follow the citation format used in existing notes
- Maintain consistent frontmatter structure across notes
- Link related concepts using wiki-links
